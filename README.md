# Portfolio Project 3 - Bike Buyers Data Analysis + Dashboard (Excel)

This project demonstrates Excel-based data analysis using a Bike Buyers dataset. The focus is on data cleaning, exploratory analysis, and dashboard creation to uncover purchasing trends.

### Key Skills Demonstrated

Data cleaning and transformation

Excel formulas

Pivot tables for analysis

Interactive dashboard design with charts and slicers

### Project Structure

Cleaned Data / Working Sheet – prepared dataset for analysis
- Removing duplicate records
- Standardising categorical values (e.g., marital status, gender)
- Formatting numeric and text fields
- Creating new calculated fields where necessary

Pivot Tables – insights by income, age, commute distance, and demographics

Dashboard – interactive visuals summarising key trends

### Key Insights

Higher income groups are more likely to purchase bikes

Age and commute distance influence buying behaviour

Demographics impact purchasing patterns

### Tools

Microsoft Excel
